# Repository instructions

- Never rename the Git branch from `master` to `main` unless the user explicitly requests it.
- This repository is the standalone public TypeScript flight-sim prototype.
- Keep public-facing copy and UI generic. Exact aircraft source/reference names may remain in engineering code and comments when they are useful for implementation accuracy.
- Keep this repository self-contained; its public build and development instructions must not require unrelated repositories.
- Desktop browsers are the current acceptance target. Preserve responsive behavior and perform occasional mobile-sized smoke checks when practical.
